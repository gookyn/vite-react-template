# vite-react-template

Vite + React + TypeScript + Antd Mobile 搭建模板项目

## 开发方法

### 安装依赖

```shell
yarn
```

### 初始化 .env.local 文件

执行命令，生成本地开发需要的 `.env.local` 文件

```shell
yarn gen:env
```

修改 `.env.local` 文件, 根据自身情况更新自定义配置（默认为开发环境的配置）

### 本地开发

```shell
yarn start
```

### 本地编译

```shell
yarn build
```
